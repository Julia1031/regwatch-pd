# RegWatch

Веб-приложение для мониторинга законодательных изменений РФ с автоматическим юридическим комментированием через локальную LLM.

## Стек

- **FastAPI** + Jinja2 — веб-интерфейс
- **SQLite** + SQLAlchemy async — база данных
- **PyMuPDF** + pytesseract — извлечение текста из PDF
- **Ollama / Qwen3:14b** — юридический анализ (локально)
- **APScheduler** — ежедневный сбор в 09:00 МСК

## Источник данных

`publication.pravo.gov.ru` — официальный интернет-портал правовой информации.
Блоки: Президент, Правительство, Федеральные органы.

## Запуск

```bash
# 1. Установить зависимости
pip install -r requirements.txt

# 2. Убедиться, что Ollama запущена с нужной моделью
ollama pull qwen3:14b
ollama serve  # если не запущена как сервис

# 3. Запустить приложение
uvicorn src.main:app --reload --host 0.0.0.0 --port 8000
```

Открыть: http://localhost:8000

## Использование

1. Нажать **«Собрать новые документы»** — сбор запустится в фоне (~2–5 мин)
2. Открыть любой документ и нажать **«Запустить анализ»** — Qwen3 сформирует юридический комментарий (~1–3 мин)
3. Фильтровать список по дате и блоку

## OCR (опционально)

Для документов-сканов нужен Tesseract:

```bash
# macOS
brew install tesseract tesseract-lang

# Ubuntu/Debian
sudo apt-get install tesseract-ocr tesseract-ocr-rus
```
